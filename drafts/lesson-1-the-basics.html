Getting Around in the Terminal

Where Am I?

Type the following to figure out where you are currently in the file system.

$ pwd
/home/username

It stands for "print working directory".

What's here?

The following shows you what files and folders are available in the current directory.

$ ls
projects  file.txt

It's short for "list".

Go Somewhere Else

To move into another directory, use the cd command followed by the name of the folder you want to move to.

$ cd projects/

It stands for "change directory".

A Simple Madlib

#!/usr/bin/env python3

exclamation = input("Type an exclamation...")
adverb = input("Type an adverb..")
noun = input("Type a noun...")
adjective = input("Type an adjective...")

madlib = (
  exclamation.capitalize() + 
  "! he said " + 
  adverb + 
  " as he jumped into his convertible " + 
  noun + 
  " and drove off with his " + 
  adjective + 
  " wife."
)

print(madlib)
